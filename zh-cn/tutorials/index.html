# 模板引擎

框架内置 egg-view 作为模板解决方案，并支持多模板渲染，每个模板引擎都以插件的方式引入，但保持渲染的 API 一致。查看如何使用模板，如果想更深入的了解，可以查看模板插件开发。

可使用以下模板引擎，更多查看

  • egg-view-nunjucks
  • egg-view-ejs
  • egg-view-handlebars
  • egg-view-pug
  • egg-view-xtpl

# 数据库

官方 ORM 模型还在设计中，但现在可以使用

  • egg-sequelize
  • egg-mongoose
  • egg-mysql，可查看 MySQL 教程
